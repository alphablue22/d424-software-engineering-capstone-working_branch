Title and purpose of the application: The Vacation Planner App helps users organize their trips and excursions. 
It allows users to add vacations, enter details like hotel names and dates, and keep track of excursions associated 
with each vacation. The app ensures smooth planning with features like date validation, alerts, and sharing capabilities.

Directions for how to operate the application: Start on the home screen, where vacations are listed. 
To add a vacation, tap the "Add Vacation" button. You can view, edit, or delete vacation details by selecting them 
from the list. Excursions can be managed within the vacation details screen by adding, updating, or deleting them. 
Alerts for vacation start and end dates, as well as excursion dates, can be set from their respective screens. 
Sharing vacation details is supported via email, SMS, or clipboard.

Android version for the signed APK: The app is compatible with Android 8.0 (API 26) and higher.
